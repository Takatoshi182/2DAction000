04_実行ファイル
このフォルダには、2Dゲーム「洞窟危機一髪～あと10秒でこの洞窟崩れるってマジですか？～」のプロジェクトが入っています。

制作期間：10月～11月半ば（約1か月半）
制作人数：1人
ジャンル：10秒アクション
工夫した点：①炎の揺らめきのタイミングをずらして無機質な動きにならないようにした。②炎、お宝、ゴールを列挙型にし、1つのファイルにまとめたこと。
改善点：ゲームシステムの設定に時間がかかったため、ステージを1つしか作れなかったこと。外部ファイルなどを用いて作業の効率化を図りたい。
